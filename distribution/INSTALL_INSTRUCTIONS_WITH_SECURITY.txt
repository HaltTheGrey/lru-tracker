HOW TO USE:
================================================================================

ADD A STATION:
  1. Click "â• Add Station"
  2. Enter station name (e.g., "Station A - Rack 12")
  3. Set minimum LRU count
  4. Set maximum LRU count
  5. Click "Add Station"

UPDATE LRU COUNT:
  1. Click on a station in the list (or select from dropdown)
  2. Enter current LRU count
  3. Click "Update LRU Count"
  4. Status updates automatically (Green/Yellow/Red)

EXPORT TO EXCEL:
  1. Click "Export to Excel"
  2. Choose "New File" or "Append to Existing"
  3. Select save location
  4. Open Excel file to view data

VIEW TRENDS:
  1. Select a station
  2. Click "Show Trend Analysis"
  3. See historical LRU usage patterns

TROUBLESHOOTING:
================================================================================

PROBLEM: Can't extract the ZIP file
SOLUTION: Right-click â†’ "Extract All..." (not just "Open")

PROBLEM: App won't run
SOLUTION: Make sure you extracted the files first, then run LRU_Tracker.exe
          from the extracted folder (not from inside the ZIP)

PROBLEM: Security warning won't go away
SOLUTION: Make sure to click "More info" first, THEN "Run anyway" appears

PROBLEM: Lost my data
SOLUTION: Your data is saved automatically at:
          C:\Users\[YourName]\AppData\Local\LRU_Tracker\lru_data.json
          Copy this file to backup your data

PROBLEM: Want to use on multiple computers
SOLUTION: Copy the lru_data.json file (see above) to the same location
          on the other computer
